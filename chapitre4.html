Chapitre 4 : L'Internet de Web


Le World Wide Web (WWW) : est un système hypertexte public sur Internet qui permet de consulter des pages accessibles sur des sites.
Le web est une partie des contenus circulant sur Internet et permet d'accéder à des documents textuels reliés entre eux par des liens hypertexte. Les trois mécanismes de base du Web sont :

  • l'URL.
  • Le protocole HTTP.
  • Le langage de description HTML.

Une page web est un document multimédia décrit élément par élément avec un langage de description HTML et stocké dans un fichier html ou .htm.
Un site web est un regroupement de pages web autour d'une thématique commune reliées entre elles par des liens hypertextes. La création d'une page web HTML nécessite l'utilisation d'éditeurs de pages web et de la connaissance de la syntaxe HTML.
